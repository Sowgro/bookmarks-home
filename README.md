# Bookmarks Home
"beyond the bar"

### Features
- Automatically grabs high quality icons as sites are visited
- Built in editor with drag and drop to move
- Customizable colors
- Configurable root folder and sort order
- Open source

### Building
1. Install libraries
```
npm install
```
2. Generate unpacked extension in `dist/`
```
npm run build
```
